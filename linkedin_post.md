🚀 **Excited to share my latest project: The Shrey Heritage Club Bespoke Portal!** 📈

I recently built an **ETF & Mutual Fund Intelligent Finder**—a comprehensive wealth planning application designed to deliver highly personalized, AI-driven investment strategies. 

I wanted a clean, professional, and secure platform where users can configure their own LLM API keys, simulate investments, and get tailored financial insights on the fly. 

Here are some of the core features I built into the portal:
🔐 **Robust Cabinet Authentication**: Secure, cross-device synchronized user sessions with SQLite, ensuring that user configurations and API keys (like Groq and Tavily) seamlessly follow you whether you login via desktop or mobile.
🤖 **AI-Powered Financial Analyst**: Integrated with advanced LLMs (via Groq and local Ollama models) to instantly analyze risk profiles, investment horizons, and custom goals, returning strict, realistic short-term and long-term expected returns.
📊 **Interactive Market Visualization**: Built dynamic, real-time charting using Recharts and `yfinance` data to track historical performance across assets like NIFTYBEES and GOLDBEES.
⚡ **Modern Tech Stack**: Powered by **Next.js**, **React**, **Tailwind CSS**, **SQLite**, and **Python** on the backend.

It was an incredible experience solving edge cases—like ensuring API keys elegantly sync across multiple devices and enforcing strict, realistic parameters on LLM prompts so the AI provides grounded financial expectations.
